Relational Algebra
===================

Binary Operators
----------------

- ``∪`` set union
- ``∩`` set intersection
- ``-`` set difference
- ``⨯`` carthesian product


Unary Operators
---------------

- ``π`` projection
- ``σ`` restriction
- ``ρ`` rename


Joins
-----

- ``⋈`` natural join
- ``θ``-join & equijoin
- ``⋉`` & ``⋊`` semijoin
- ``▷`` antijoin
- ``÷`` division
